1.可以用 in /of

2.養成習慣: 要加 key

3.不建議在同一元素同時使用 for 跟 if
for 優先級比 if 高

數字列表: v-for="item in dataList"

  • {{ item }}

使用者列表: v-for="(user, index) in users"

  • {{ index }} - {{ user.name }} - age: {{ user.age}}

數字: v-for="number in 5"

  • {{ number }}

computed: v-for="number in evenNumbers"

  • {{ number }}

簡單篩選:

  • {{ number }}
